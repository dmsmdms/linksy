Создай мне подробное техническое задание Telegram Mini App под названием Linksy.
Для Frontend используй PUG, SASS, typescript.
Бэкенд напиши на языке Python асинхронный в один поток. Комуникация через REST API в JSON подобно Telegram Bot API. В качесте базы данных используй sqlite. Минимизируй износ диска.
Придложение будет работать как одностраничный сайт. Навигация будет происходить при помощи TypeScript и изменения видимости секций.
Переход по к примеру в профиль будт происходить следующим образом: /index.html#profile.
В качестве веб сервера для статического контента будет использоваться Nginx.
Для получения обновленией будет использоваться WebSocket соединение.
Так же нужна админ панель для мониторинга событий и управления пользвателями.
Будет возможность получать Push уведомления через Telegram Bot.
Сборка проекта будет целиком на meson. Для сборки PUG используется команда pug. Для SCSS используется sassc. Для typescript используется tsc.
При открытии приложение сразу разворачивается на весь экран.
Добавь возможность блокировки сервиса на время технических работ.
Авторизация через Telegram OAuth.
Все пользователи приложения равноправные. Админы работают через админ панель.

Панель навигации:
1.5. Панель навигации прикреплена всегда к низу экрана и содержит следующие пункты, между которыми можно переключаться:
1.5.1. События
1.5.2. Ранг
1.5.3. Создать событие
1.5.4. Чат
1.5.5. Профиль

Основные экраны:
1. События (панель навигации отображается)
1.1. Верхний левый угол - значёк фильтров (под значком фильтров в кружочке пишется количество активных фильтров)
1.2. Верхний правый угол - значёк уведомлений (под значком уведомлений в кружочке пишется количество уведомлений)
1.3. Блок историй:
1.3.1. Первая история - это история текущего пользователя. Отображается его аватарка и значёк +
1.3.2. Дальше истории других пользователей
1.4. Следующая строка - дни мероприятий: Все, Сегодня, Завтра, Выходные, Неделя, Месяц. Рядом с днями мероприятий пишется количество событий в этот день.
1.5. Блок карт событий следующего содержания:
1.5.1. Дата события (Sun 15 Feb, 11:00-12:00)
1.5.2. Расстояние до события (5 km)
1.5.3. Название события (шрифт больше)
1.5.4. Кружочки с аватарками участников события до 5 штук. Остальные участники будут просто отображаться в виде отдельноего кружочка с текстом +14 (оставшееся количество участников)
1.5.5. Стоимость мероприятия
1.5.6. Количество лайков
1.5.7. На фоне отображается фотография события

2. Фильтры (панель навигации скрыта)
2.1. Категории событий. Можно выбирать несколько. В начале есть выбор "Все категории". Отображение - иконка + название. При выборе иконка превращается в галочку.
2.2. Языки событий. Можно выбирать несколько. В начале есть выбор "Все языки". Отображение - флаг + название. При выборе иконка превращается в галочку.
2.3. Город или дистанция от местоположения.

3. Нотификации (панель навигации скрыта)
2.1. Правый верхний угол - настройка нотификаций.
2.2. Групировка нотификай по дням (сегодня, последние 7 дней, последний месяц). Если сегодня или на неделе не было нотификаций, то скрываем пункты.
2.3. Нотификации:
2.3.1. Аватарка источника нотификации. Снизу в кружочке отображается иконка типа нотификации (календать, галочка, пользователь)
2.3.2. Текст описание нотификации
2.3.3. Дата нотификации (05 Feb 2026 11:26)

4. Ранг (панель навигации отображается)
1.1. Левый верхний угол - значёк вопроса в кружочке
1.2. Правый верхний угол - количество звёзд в профиле
1.3. По центру большой цифрой отображается ранг пользователя
1.4. Шкала XP до следующего повышения ранга
1.5. Левая кнопка - Текущие задания. Правая кнопка - выполненные задания
1.6. Карты заданий:
1.6.1. Иконка залания (левый верхний угол)
1.6.2. Количество XP и звёзд за данное задание (правый верхний угол)
1.6.3. Описание задания
1.6.4. Шкала выполнения задания

5. Создать событие (панель навигации отображается)
5.1. Название события
5.2. Количество учатников
5.3. Язык события
5.4. Кактегория события
5.5. Локация события
5.6. Дата и время
5.7. Цена события
5.8. Изображение события
5.9. Описание события
5.10. Гедер события (Все, Мужчины, Женщины)
5.11. Возраст события
5.12. Создать приватное событие
5.13. Опубликовать

6. Настройка нотификаций (панель навигации скрыта). Просто содержит список групированных переключателей. Каждый переключатель соджжит короткое название и полноценное описание внизу.
6.1. События
6.1.1. Получать новые события от пользователей на которых вы подписаны
6.1.2. Напоминание перед событием (за пару часов, за день, никогда)
6.1.3. Уведомление когда моё событие получило лайк
6.2. Чаты
6.2.1. Хозяин чата написал сообщение
6.2.2. Упомянание в сообщении
6.2.3. Другие сообщение в чате
6.3. Личные сообщения
6.3.1. Нотификация при личном сообщении

7. Профиль (панель навигации отображается)
7.1. Левый верхний угол - настройки
7.2. Правый верхний угол - уведомления
7.3. Аватарки профиля (можно листать аватарки свайпом)
7.4. Имя пользователя, Возраст
7.5. Количество подписчиков, подписок, друзья
7.6. Строка событий (предстоящие, организованные, завершённые)
7.7. Карты событий

8. Настройки (панель навигации скрыта)
8.1. Редактировать профиль
8.2. Конфиденциальность
8.3. Настройка нотификаций
8.4. Язык приложения
8.5. Обратная связь
8.6. Удалить аккаунт
